1. nodejs 설치 및 버전 확인

```
 node -v
```

2. react.js 설치

```
npm install -g create-react-app
create-react-app --version
```

3. 프로젝트 생성

```
create-react-app project-name
```

4. web publish start with create-react-app

```
npm start
```

  - 자동 새로 고침
