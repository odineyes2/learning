# hosting

## 1. hosting

- firebase console > build > hosting > get started
- 이하 안내에 따라 절차 진행
- install firebase cli
- $ npm install -g firebase-tools
- $ firebase login
- $ firebase init
- hosting 선택
- use an existing project
- $ npm run build
- 생성된 폴더 확인 : dist
- dist
- single page? : yes
- firebase.json 파일 확인

## 2. package.json 수정

```json
"scripts": {
    "predeploy": "npm run dev"
    "deploy": "firebase deploy"
}
```

- $ npm run deploy
- 사이트 접속 확인
- 파이어베이스 콘솔 확인

## 3. 코드 수정 후 반영 하기

- $ npm run deploy
