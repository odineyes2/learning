## 식별자 네이밍 규칙

식별자는 특수문자를 제외한 숫자, 언더스코어, 달러기호를 포함할 수 있다.
특수문자를 제외한 문자, 언더스코어(_), 달러기호($)로 시작해야 한다.
숫자로 시작하는 것은 불허한다.
예약어는 식별자로 사용할 수 없다.

예약어
await
break
...

## 카멜 케이스(변수, 함수)
```
const firstName;
```

## 파스칼 케이스(생성자 함수, 클래스 이름)
```
const FirstName;
```

## 스네이크 케이스
```
const first_name;
```

## 헝가리언 케이스
```
const strFirstName;
```

## 일반적인 원칙
- 변수나 함수의 이름에는 카멜 케이스를 사용한다.
- 생성자 함수, 클래스의 이름에는 파스칼 케이스를 사용한다.
