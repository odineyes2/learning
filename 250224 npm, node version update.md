# version update

## npm 버전 확인

```sh
-$ npm --version
```

## node 자체의 버전 업데이트

```sh
-$ npm install -g npm  #LTS 로 설치
```

## npm 자체의 버전 업데이트

```sh
-$ npm install -g npm
```

## 모든 글로벌 패키지 업데이트

```sh
-$ npm update -g
```

## 프로젝트 내 패키지 업데이트

```sh

-$ npm update

```

## 메이저 업데이트 확인 및 적용

```sh
npm outdated
npm update         # 가능한 업데이트 적용
npx npm-check -u   # 대화형 업데이트 (설치 필요: `npm install -g npm-check`)
```
